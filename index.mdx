---
title: "LLVM Compiler Infrastructure"
description: "A collection of modular, reusable compiler and toolchain technologies for building high-performance compilers, optimizers, and code generation backends."
---

LLVM is the industry-standard compiler infrastructure used by Apple, Google, AMD, ARM, and many others. It provides a powerful intermediate representation (IR), a rich suite of optimization passes, and target-independent code generation for architectures including x86, ARM, RISC-V, AMDGPU, and more.
